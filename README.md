# 📸 InstaClone Backend (Spring Boot REST API)

🚀 A scalable RESTful backend for a social media platform built with Spring Boot, implementing real-world features like posts, likes, comments, and follow systems.

---

## 📌 About the Project

This project is a scalable backend system for a social media platform developed using Java and Spring Boot. It is designed to efficiently manage user interactions and simulate real-world social networking features through a well-structured backend architecture.

The application allows users to create posts, interact with content, and connect with other users. Core functionalities include user registration, post creation, liking posts, commenting, and managing follower relationships (follow/unfollow), replicating key features of modern social media platforms.

For data persistence, the system integrates Spring Data JPA with a SQL database hosted on Supabase, enabling efficient storage and retrieval of user data, posts, and interactions. The application uses JSON for request-response handling and RESTful APIs for communication between client and server. All APIs were tested and validated using Postman.

The project follows Object-Oriented Programming (OOP) and SOLID design principles, ensuring modular, maintainable, and scalable code. A layered architecture (Controller, Service, Repository) is implemented to separate concerns and improve code organization.

## 🧱 Project Architecture

This project follows a layered architecture:

- **Controller Layer** → Handles HTTP requests and API endpoints  
- **Service Layer** → Contains business logic  
- **Repository Layer** → Handles database operations using JPA  
- **Model Layer** → Entity classes mapped to database
